# CLH 売上管理サイト（メーカー売り上げ伝票サイト）

## 概要
カーロックホームズ株式会社の売上管理ダッシュボード。PDF伝票（請求書・納品書・注文書）をAI解析して登録し、セクション別に売上を管理する。

## 技術スタック
- React + Vite
- pdfjs-dist（PDF解析）
- xlsx / SheetJS（Excel出力）
- CSS アニメーション（カウントアップ、フェードイン、スライドイン、チャートグロウ、プログレスバー、パルス等）

## 詳細な実装内容

### ダッシュボード
- **統計カード**: 美和ロック、ゴール、シブタニ、管理会社等、合計の売上表示
- **ドーナツチャート**: セクション別売上比率の可視化
- **月別売上推移**: バーチャートで過去12ヶ月の推移を表示
- **高額案件Top5**: 金額が高い案件を5件表示（詳細モーダル対応）
- **最近の伝票**: 登録順に表示、詳細表示対応
- **サマリーバー**: 回収率をプログレスバーで表示（アニメーション付き）

### セクション別ページ（SectionPage.jsx）
- 各セクション（美和ロック/ゴール/シブタニ/管理会社等）の詳細統計表示
- セクション別の月別売上推移（バーチャート）
- セクション内の伝票一覧（ソート・詳細表示対応）

### PDF登録フロー（PDFUpload.jsx）
- **ステップ1**: ドラッグ&ドロップでPDFアップロード
- **ステップ2**: pdfjs-distでPDF解析、テキスト自動抽出
- **ステップ3**: 抽出データの確認・編集フォーム
- **ステップ4**: データ登録とトースト通知

### 全伝票一覧（RecordsList.jsx）
- **検索**: キーワード検索（複数ワード対応）
- **フィルタ**: カテゴリ別（美和ロック/ゴール/シブタニ/管理会社等）、ステータス別フィルタ
- **ソート**: 金額、日付、セクションで昇降順ソート
- **詳細表示**: モーダルで全情報表示
- **削除機能**: 伝票削除機能（確認ダイアログ付き）

### データ出力（ExportPage.jsx）
- **Excel形式**: .xlsx ファイル出力（複数シート対応）
- **CSV形式**: Google スプレッドシート互換形式でエクスポート
- エクスポート時にフォーマット自動整形

### アニメーション実装
- **カウントアップ**: 統計カード、金額表示のアニメーション
- **フェードイン**: コンポーネント読み込み時のフェード効果
- **スライドイン**: モーダル、サイドバーのスライド動作
- **チャートグロウ**: グラフ描画時の段階的表示
- **プログレスバー**: 回収率表示のアニメーション
- **パルス効果**: インタラクティブ要素のホバー時動作

### レスポンシブデザイン
- **PC（1200px以上）**: 5カラムグリッド、サイドバー常時表示
- **タブレット（768px～1200px）**: 2カラムグリッド、サイドバー開閉機能
- **スマートフォン（768px未満）**: 1カラムレイアウト、全画面表示対応

### 状態管理（store.jsx）
- Context API + useReducer によるグローバル状態管理
- レコード追加・更新・削除のアクション実装
- ローカルストレージと連携

### ユーティリティ関数（utils.js）
- PDF解析関数（pdfjs-dist利用）
- データフォーマット関数
- Excel/CSV エクスポート関数
- テキスト抽出・処理関数

### カスタムフック（hooks.js）
- `useAnimatedNumber`: 数値カウントアップアニメーション
- `useInView`: スクロール時の表示検出
